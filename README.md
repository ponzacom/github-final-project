# Simple Interest Calculator

A simple Python-based calculator for computing simple interest on loans or investments.

## Description

This project calculates simple interest using the formula:
```
Simple Interest = (Principal × Rate × Time) / 100
```

## Features

- Calculate simple interest with principal, rate, and time inputs
- Easy-to-use command-line interface
- Input validation
- Clear output formatting

## Requirements

- Python 3.x

## Installation

1. Clone this repository or download the files
2. Ensure Python 3 is installed on your system

## Usage

```bash
python simple_interest_calculator.py
```

### Example

```
Enter Principal Amount: 10000
Enter Rate of Interest (% per annum): 5
Enter Time Period (in years): 2

Simple Interest: 1000.00
Total Amount: 11000.00
```

## Formula

- **Simple Interest (SI)** = (P × R × T) / 100
- **Total Amount** = P + SI

Where:
- P = Principal Amount
- R = Rate of Interest per annum
- T = Time period in years
